# Task 01.1 - First Time Git Setup & Initial Commit

## Goal

Set up Git, create a GitHub account, and push your Smart Task Manager project to a remote repository.

By the end of this task, your project will be safely stored on GitHub with version control enabled.

---

## 🧠 Why This Matters

* Tracks your code changes over time
* Lets you revert mistakes
* Shows your work to recruiters
* Enables collaboration

---

## Step 1: Install Git

1. Go to:
   https://git-scm.com/downloads

2. Download Git for Windows

3. Run the installer:

   * Keep **default settings**
   * Click **Next → Next → Finish**

---

## Step 2: Verify Git Installation

Open **Command Prompt** or **Terminal** and run:

```bash
git --version
```

You should see something like:

```text
git version 2.x.x
```

---

## Step 3: Configure Git (First Time Only)

Set your name:

```bash
git config --global user.name "Your Name"
```

Set your email:

```bash
git config --global user.email "your-email@example.com"
```

👉 Use the same email you will use for GitHub

---

## Step 4: Create a GitHub Account

1. Go to:
   https://github.com

2. Click **Sign up**

3. Enter:

   * Email
   * Password
   * Username

4. Verify your account

---

## Step 5: Create a New Repository on GitHub

1. Click **+ (top right)** → **New repository**

2. Enter:

   * Repository name: `smart-task-manager`
   * Visibility: Public (recommended)

3. ❗ IMPORTANT:

   * Do NOT check:

     * Add README
     * Add .gitignore

4. Click **Create repository**

---

## Step 6: Initialize Git in Your Project

Open terminal inside your project folder:

```bash
cd D:\Smart_Task_Manager
```

Run:

```bash
git init
```

---

## Step 7: Add .gitignore

Run:

```bash
dotnet new gitignore
```

This prevents unnecessary files from being tracked.

---

## Step 8: Add Files to Git

```bash
git add .
```

---

## Step 9: Create First Commit

```bash
git commit -m "Initial project setup with clean architecture structure, Swagger and OpenAPI configured"
```

---

## Step 10: Connect to GitHub Repository

Copy your repo URL from GitHub, then run:

```bash
git remote add origin https://github.com/<your-username>/smart-task-manager.git
```

---

## Step 11: Push Code to GitHub

```bash
git branch -M main
git push -u origin main
```

---

## Step 12: Verify

Go to your GitHub repository page.

You should see:

* Your project files
* Folder structure
* Commit message

---

## 🎉 Done!

You have successfully:

* Installed Git
* Created a GitHub account
* Initialized version control
* Made your first commit
* Pushed code to GitHub

---

## 🧠 Important Notes

* Commit often (after each major step)
* Use meaningful commit messages
* Never commit:

  * bin/
  * obj/
  * secrets
